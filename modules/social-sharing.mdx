---
title: "Social sharing"
description: "Share product and content links across X, Facebook, Pinterest, and via copy-link in 86d."
---

The social-sharing module provides share buttons for X, Facebook, Pinterest, WhatsApp, email, and copy-link. Each share emits a `social.shared` event so you can attribute traffic.

**Source:** [`modules/social-sharing`](https://github.com/86d-app/86d/tree/main/modules/social-sharing) · **npm:** [`@86d-app/social-sharing`](https://www.npmjs.com/package/@86d-app/social-sharing)

## Installation

```sh
86d module add social-sharing
```

## Components

- `<ShareButtons url={url} />`, `<CopyLinkButton url={url} />`.
