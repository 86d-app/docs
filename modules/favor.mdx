---
title: "Favor"
description: "Last-mile delivery via Favor for 86d. Texas-focused on-demand delivery integration with rate quoting and webhook status."
---

The Favor module integrates with the Favor Delivery API for on-demand local delivery in Favor's coverage area (primarily Texas). Like the other delivery modules, it quotes at checkout, creates a delivery on order placement, and receives status webhooks.

**Source:** [`modules/favor`](https://github.com/86d-app/86d/tree/main/modules/favor) · **npm:** [`@86d-app/favor`](https://www.npmjs.com/package/@86d-app/favor)

## Installation

```sh
86d module add favor
```
