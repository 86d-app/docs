description: "Sync your 86d catalog to Facebook Shops via the Meta Commerce API. Manage commerce from your admin sidebar."
---

The Facebook Shop module pushes your catalog to Facebook Shops. Catalog sync, inventory updates, and order ingestion happen via the Meta Commerce API.

**Source:** [`modules/facebook-shop`](https://github.com/86d-app/86d/tree/main/modules/facebook-shop) · **npm:** [`@86d-app/facebook-shop`](https://www.npmjs.com/package/@86d-app/facebook-shop)

## Installation

```sh
86d module add facebook-shop
```
