---
title: "Redirects"
description: "URL redirect management for an 86d storefront. Permanent and temporary, exact and pattern-based. Preserves SEO during URL changes."
---

The redirects module manages 301 and 302 redirects from old URLs to new ones. Useful when migrating from another platform, restructuring URLs, or sunsetting product pages.

**Source:** [`modules/redirects`](https://github.com/86d-app/86d/tree/main/modules/redirects) · **npm:** [`@86d-app/redirects`](https://www.npmjs.com/package/@86d-app/redirects)

## Installation

```sh
86d module add redirects
```

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/admin/redirects` | List |
| `POST` | `/admin/redirects` | Create |
| `PUT` | `/admin/redirects/:id` | Update |
| `DELETE` | `/admin/redirects/:id` | Delete |
