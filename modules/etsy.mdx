---
title: "Etsy"
description: "Etsy shop sync for 86d. Push listings, manage variations, and pull Etsy orders into your 86d order queue."
---

The Etsy module syncs your catalog to your Etsy shop via the Etsy v3 API. It manages listing variations, inventory, and pulls Etsy orders into `@86d-app/orders` with `channel: "etsy"`.

**Source:** [`modules/etsy`](https://github.com/86d-app/86d/tree/main/modules/etsy) · **npm:** [`@86d-app/etsy`](https://www.npmjs.com/package/@86d-app/etsy)

## Installation

```sh
86d module add etsy
```

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/admin/etsy/listings` | List Etsy listings |
| `POST` | `/admin/etsy/listings/sync` | Push catalog |
