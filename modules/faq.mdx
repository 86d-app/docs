title: "FAQ module"
description: "Self-service knowledge base with search for an 86d storefront. Group questions, embed on product pages, and surface in support flows."
---

The FAQ module manages a self-service knowledge base. Questions can be grouped (Shipping, Returns, Account) and embedded contextually on product detail pages or as a standalone `/faq` page. Search is built in.

**Source:** [`modules/faq`](https://github.com/86d-app/86d/tree/main/modules/faq) · **npm:** [`@86d-app/faq`](https://www.npmjs.com/package/@86d-app/faq)

## Installation

```sh
86d module add faq
```

## Store endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/faq` | All approved questions, grouped |
| `GET` | `/faq/search?q=` | Search the FAQ |

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `POST` | `/admin/faq` | Create |
| `PUT` | `/admin/faq/:id` | Update |

## Components

- `<FaqList />`, `<FaqGroup slug={slug} />`, `<FaqSearch />`.
