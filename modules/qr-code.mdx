---
title: "QR code"
description: "Generate and track QR codes for 86d. Encode product, collection, or campaign URLs and capture per-scan analytics."
---

The qr-code module generates QR codes for any URL on your store and tracks scans with attribution metadata (campaign, source, medium). Useful for in-store signage, packaging inserts, and offline campaigns.

**Source:** [`modules/qr-code`](https://github.com/86d-app/86d/tree/main/modules/qr-code) · **npm:** [`@86d-app/qr-code`](https://www.npmjs.com/package/@86d-app/qr-code)

## Installation

```sh
86d module add qr-code
```

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `POST` | `/admin/qr-code` | Create a QR code |
| `GET` | `/admin/qr-code/:id/analytics` | Scan analytics |
