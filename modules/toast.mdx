---
title: "Toast POS"
description: "Toast POS integration for 86d. Sync menu, route online orders to Toast, and reconcile in-store and online sales."
---

The Toast module connects an 86d store to a Toast POS for restaurants and cafes. The catalog syncs as a menu, online orders route to Toast for in-restaurant fulfillment, and Toast in-store sales can flow back into `@86d-app/orders` for unified reporting.

**Source:** [`modules/toast`](https://github.com/86d-app/86d/tree/main/modules/toast) · **npm:** [`@86d-app/toast`](https://www.npmjs.com/package/@86d-app/toast)

## Installation

```sh
86d module add toast
```

## Configuration

Set the `TOAST_*` environment variables (`TOAST_API_KEY`, `TOAST_RESTAURANT_GUID`, `TOAST_SANDBOX`). See [Environment variables](/configuration/environment-variables).
