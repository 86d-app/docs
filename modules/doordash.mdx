---
title: "DoorDash"
description: "Last-mile delivery via DoorDash Drive for 86d. Quote rates, dispatch deliveries, and track status from the admin."
---

The DoorDash module integrates with the DoorDash Drive API for on-demand local delivery. The store quotes rates at checkout, dispatches a delivery on order placement, and receives webhook updates as the courier moves through pickup and dropoff.

**Source:** [`modules/doordash`](https://github.com/86d-app/86d/tree/main/modules/doordash) · **npm:** [`@86d-app/doordash`](https://www.npmjs.com/package/@86d-app/doordash)

## Installation

```sh
86d module add doordash
```

## Configuration

```ts
doordash({
  developerId: "...",
  keyId: "...",
  signingSecret: "...",
  sandbox: "true",
});
```

## Store endpoints

| Method | Path | Description |
| --- | --- | --- |
| `POST` | `/doordash/quote` | Quote a delivery rate |
| `POST` | `/doordash/webhooks` | Receive delivery status webhooks |

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/admin/doordash/deliveries` | Active and completed deliveries |
| `POST` | `/admin/doordash/deliveries/:id/cancel` | Cancel |
