---
title: "Wishlist"
description: "Save and track favorite products in 86d. Per-customer wishlists with shareable links and stock-back notifications."
---

The wishlist module lets shoppers save products for later. Wishlists are per customer, optionally shareable via a public link, and can trigger notifications when a wishlisted item drops in price or restocks.

**Source:** [`modules/wishlist`](https://github.com/86d-app/86d/tree/main/modules/wishlist) · **npm:** [`@86d-app/wishlist`](https://www.npmjs.com/package/@86d-app/wishlist)

## Installation

```sh
86d module add wishlist
```

## Configuration

```ts
wishlist({
  notifyOnPriceDrop: "true",
  notifyOnRestock: "true",
  publicShareEnabled: "true",
});
```

## Store endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/wishlist/me` | Get the customer's wishlist |
| `POST` | `/wishlist/me/items` | Add an item |
| `DELETE` | `/wishlist/me/items/:id` | Remove an item |
| `GET` | `/wishlist/share/:token` | Public read of a shared wishlist |

## Components

- `<WishlistButton productId={id} />`: heart toggle on product cards.
- `<WishlistPage />`: customer self-service wishlist view.
