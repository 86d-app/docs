---
title: "Price lists"
description: "Customer-segment or channel-specific pricing for 86d. Override product prices for B2B groups, regions, or specific channels."
---

The price-lists module defines per-segment price overrides. A price list applies to one or more customer groups (B2B, wholesale, employees) or to specific sales channels and overrides product prices accordingly. Multiple price lists can stack with explicit precedence.

**Source:** [`modules/price-lists`](https://github.com/86d-app/86d/tree/main/modules/price-lists) · **npm:** [`@86d-app/price-lists`](https://www.npmjs.com/package/@86d-app/price-lists)

## Installation

```sh
86d module add price-lists
```

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/admin/price-lists` | List |
| `POST` | `/admin/price-lists` | Create |
| `PUT` | `/admin/price-lists/:id` | Update |
| `POST` | `/admin/price-lists/:id/items` | Add product overrides |
