---
title: "Vendors"
description: "Multi-vendor marketplace support for 86d. Each vendor has their own catalog slice, payouts, and admin access."
---

The vendors module enables marketplace-style multi-vendor stores. Each product is owned by a vendor; orders split into per-vendor sub-orders for fulfillment; the platform takes a configurable commission and pays vendors out on schedule.

**Source:** [`modules/vendors`](https://github.com/86d-app/86d/tree/main/modules/vendors) · **npm:** [`@86d-app/vendors`](https://www.npmjs.com/package/@86d-app/vendors)

## Installation

```sh
86d module add vendors
```
