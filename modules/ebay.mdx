---
title: "eBay"
description: "eBay listing management for 86d. Push catalog as listings, sync inventory, and ingest sold orders into your 86d order queue."
---

The eBay module syncs your catalog to eBay listings via the eBay Sell API. Sold orders flow into the `@86d-app/orders` queue tagged with `channel: "ebay"`.

**Source:** [`modules/ebay`](https://github.com/86d-app/86d/tree/main/modules/ebay) · **npm:** [`@86d-app/ebay`](https://www.npmjs.com/package/@86d-app/ebay)

## Installation

```sh
86d module add ebay
```

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/admin/ebay/listings` | List active listings |
| `POST` | `/admin/ebay/listings/sync` | Push catalog updates |
