---
title: "Preorders"
description: "Sell products before they are available for fulfillment. Capture payment up front and ship when stock arrives."
---

The preorders module marks selected products as available for preorder during a configurable window. Customers can purchase before inventory exists; the module flags each preorder line item, captures payment if you choose, and tracks expected ship date.

**Source:** [`modules/preorders`](https://github.com/86d-app/86d/tree/main/modules/preorders) · **npm:** [`@86d-app/preorders`](https://www.npmjs.com/package/@86d-app/preorders)

## Installation

```sh
86d module add preorders
```

## Configuration

```ts
preorders({
  capturePaymentUpFront: "true",
  defaultLeadTimeDays: "14",
});
```

## Admin endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/admin/preorders` | Active preorder campaigns |
| `POST` | `/admin/preorders` | Create a campaign |
| `POST` | `/admin/preorders/:id/release` | Convert preorders to fulfillable orders |
